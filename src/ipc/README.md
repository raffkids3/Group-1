# Inter-Task Communication (Student Work Area)

## Purpose

This directory contains message queue and IPC logic introduced in **Module 6**.

IPC enables structured coordination between tasks.

---

## What Students Should Do

- Define message structures (see `project.h`)
- Implement producer and consumer tasks
- Choose appropriate queue sizes and blocking behavior

---

## Evidence Expectations

- Output showing messages sent and received
- Notes explaining queue sizing and timeout decisions

# Module 2 – Guided Coding Lab Checklist

## A. Environment Setup
- [ ] I can configure the project in VS Code (CMake Tools).
- [ ] I can build without errors.
- [ ] I can run the executable locally.

## B. Build + Run Verification
- [ ] The program prints startup banner text.
- [ ] I see Heartbeat output lines.
- [ ] I see Worker output lines.
- [ ] The program exits with: `Baseline OK.`

## C. RTOS Configuration Familiarity
- [ ] I located `rtos/FreeRTOSConfig.h`.
- [ ] I can point to the tick rate setting.
- [ ] I can point to the max priorities setting.
- [ ] I can explain how tick rate affects delay granularity (high level).

## D. GitHub Baseline
- [ ] I cloned the team repo.
- [ ] I integrated this baseline.
- [ ] I made at least 2 meaningful commits (not one massive commit).
- [ ] I pushed to remote and confirmed teammates can build.
